Real-Time Macro Impact Tracker

Analyzes how macroeconomic data releases impact equities, rates, FX, and volatility in real time using intraday data.

## Features
- Dynamic macro event date generation with weekday adjustment
- Full UTC timezone consistency
- Robust handling of empty data and event windows
- Measures market impact at 15-minute and 60-minute horizons

## Assets
- SPY (Equities)
- TLT (Rates)
- UUP (FX)
- VIXY (Volatility)

## Outputs
- Cross-asset surprise impact table
- Saved event-reaction plots:
  - market_reaction_CPI_Release_YYYYMMDD_HHMM.png
  - market_reaction_Employment_(NFP)_YYYYMMDD_HHMM.png

## Use Case
Replicates how macro and multi-asset desks analyze information flow and short-term market reactions to economic surprises.
